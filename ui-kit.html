Color tokens

Three accent meanings, never mixed: GOLD = a light is on · CYAN = another device active · IRIS = interactive/brand. Status colors: success / error / warning / info.

Buttons

A bare <button> is NEUTRAL — pick a variant. One filled primary per group; the companion is outlined. Full-width by default (touch-first); .btn-inline sizes to content.

Forms

.form-group = label over control. Inputs never go below 16px font on phones (iOS zoom).

Panels, rows, pills

.panel is the resting content card (flat + border + lift — floating layers use --shadow-modal instead). Rows inside sit recessed on .panel-row.

Device tiles

The whole tile is the switch, and a tile earns its size: tap-is-the-whole-story devices (light / door / blinds / dimmable / satellite) add .compact — a one-row plate that grows an extra row only for a live control (a lit dimmable's slider, the satellite's mic chip). .cat-<category> + .on drive the fill: gold = lit light, cyan = other device active. .manual = dashed border + lock badge. Channel controls: chip / slider / stepper / readout.
